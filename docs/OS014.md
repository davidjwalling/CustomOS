### Project os.014
Source: [os.014/os.asm](../os.014/os.asm)

### Features and Topics

### [Virtual](VIRTUAL.md) Machine Operation
- Update the Virtual Machine configuration to use os.014/os.dsk as the diskette image.
- Start the Virtual Machine.

<img src="../images/os014_VirtualBox_001.PNG" width="640"/>

### [Physical](PHYSICAL.md) Machine Operation
- Overwrite os.com on the OS boot diskette with os.014/os.com.
- Insert the OS boot diskette into the physical system's floppy disk drive A:.
- Start the system.
